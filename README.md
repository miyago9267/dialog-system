# dialogtest

Minecraft 1.20.4 劇情執行系統，命名空間：`dialogtest`。

以**多軌時間軸**驅動劇情播放，類似影片剪輯軟體的多軌道結構——台詞、動畫、移動、控制各走獨立軌道，在同一個場景時鐘上各自觸發。
